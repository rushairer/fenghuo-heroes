# 三国志列传：群英新篇

一款使用 Phaser + TypeScript + Vite 开发的三国题材 Web 策略游戏。

项目目标是从怀旧回合策略游戏的体验出发，复刻“标题初始化、选择剧本、选择势力、全国地图、视察情况、内政/外交/军事、行军、攻城、单挑、月令循环”的核心结构，同时保证代码、美术、音乐、文案和数值设计全部原创。

## 当前状态

- 标题、初始设定、君主选择、全国地图流程。
- 多势力、多城池、道路关系、城池归属变化。
- 视察情况月：主命令只保留内政、外交、军事三大栏目；人事调动、任命、出征等功能收拢进对应栏目。
- 内政：开发、调动、情报、福利、任命、税率、教育、运输。
- 外交：同盟、计策、情报、借款、还款。
- 军事：征兵、武器、人材、防卫、训练、出征。
- 行军月：远征军拥有主将、随军武将、兵力、粮草、士气、路线、移动点和状态；攻击会先进入攻城层。
- 攻城层：强攻、围城、火计、挑战、会战、撤退；挑战已进入可操作单挑界面。
- 月令：资源产出、敌方行动、随机事件、月令报告。
- 原创标题图、战场图、武将头像与程序化占位配乐。

## 本地开发

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

## 设计文档

- `DEVELOPMENT_PLAN.md`：开发计划和复刻边界。
- `ORIGINAL_GAME_STUDY.md`：原版系统复盘、已查证事实和待验证清单。
- `REPLICATION_BLUEPRINT.md`：状态机、菜单矩阵、行军/攻城/单挑的复刻实施蓝图。
- `DESIGN_DATA.md`：势力、城池、武将等数据设计。
- `ASSET_PROMPTS.md`：原创素材生成提示词记录。

- 三国势力、城池、武将和剧本数据。
