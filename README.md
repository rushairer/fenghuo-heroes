# 三国志列传：群英新篇

一款使用 Phaser + TypeScript + Vite 开发的三国题材 Web 策略游戏。

项目目标是从怀旧回合策略游戏的体验出发，复刻“标题初始化、选择剧本、选择势力、全国地图、城池命令、内政/外交/军事、人事调动、月令循环、战斗结算”的核心结构，同时保证代码、美术、音乐、文案和数值设计全部原创。

## 当前状态

- 标题、初始设定、君主选择、全国地图流程。
- 多势力、多城池、道路关系、城池归属变化。
- 城池绑定命令：内政、外交、军事、人事、机能、月令。
- 内政：开发、征兵、训练、治安、征粮。
- 外交：按邻接势力选择对象，同盟、侦察、离间、劝降。
- 军事：从当前城池选择邻接目标，下达出征令，进入军事行动，战后回写兵力、城防、缴获和城池归属。
- 人事：武将归属城池，任命、登用、移动和本城出战。
- 月令：资源产出、敌方行动、随机事件、月令报告。
- 原创标题图、战场图、武将头像与程序化占位配乐。

## 本地开发

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

## 设计文档

- `DEVELOPMENT_PLAN.md`：开发计划和复刻边界。
- `DESIGN_DATA.md`：势力、城池、武将等数据设计。
- `ASSET_PROMPTS.md`：原创素材生成提示词记录。

- 三国势力、城池、武将和剧本数据。
